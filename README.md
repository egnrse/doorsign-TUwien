# Doorsign Template for TU Wien

To use (tldr):
- fill out the [doorsign template](./doorsign_template.tex)
- compile it with `XeLatex` or `LuaLaTeX` (needed for the fonts)
- print the pdf you created
- cut on the cutlines (and the corners)
